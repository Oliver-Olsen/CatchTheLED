void click_detected ( int * prev,
int * points,
int * misses )

If a click is detected, a check will be run to verify if the 3rd LED was lit at the moment. If the 3rd LED was lit, a point is added and the blink_patternPoints(), wil animate a point scored. If the 3rd LED was not lit, the misses counter will be incremented, and the LED's will turn off for 1 second.

Parameters
prevPreviously lit LED (which here is the current one)
pointsPoint counter
